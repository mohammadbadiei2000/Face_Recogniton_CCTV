# Dataset Structure

This project uses the following dataset structure:

├── dataset/                        # Main dataset (train/val/test split)
│   ├── train/
│   ├── val/
│   └── test/
│
├── verify_face_database/          # Known face database for recognition
│   ├── person_1/
│   │   ├── 1.jpg
│   │   ├── 2.jpg
│   │   └── ...
│   ├── person_2/
│   ├── person_3/
│   └── ...
